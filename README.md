# RepQuest



RepQuest is an Android-native application that allows users to enjoy exercising and living a healthier lifestyle. Its purpose is to provide a fun, motivating experience by gamifying the workout process with simple tools that streamline the process for individuals of any experience level. The application allows users to create custom workout templates, track their workouts in real-time, earn experience points by maintaining streaks, hitting personal records, and completing sessions, and set default units in the settings menu. Additionally, users will be able to view their progress, set goals, track progress, and adjust rest periods between exercises.
